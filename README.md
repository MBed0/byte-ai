</h> Installation: <h>
1. Download
2. Download the libraries from the requirements file
3. Change the "Enter your password" section in the .env file to your own unique password
4. Open cmd and run the command "python main.py"
5. In the window that opens, enter the password you set in step 3, and you are now ready to chat and write code. If you need help, use the "help" command.
