This is an AI that combines a "chatbot + software" and is developed by Byte Comp.

</h> Installation: <h>
1. Download
2. Download the libraries from the requirements file
3. Change the "Enter your password" section in the .env file to your own unique password
4. Open cmd and run the command "python main.py"
5. In the window that opens, enter the password you set in step 3, and you are now ready to chat and write code. If you need help, use the "help" command.


Bu, Byte Comp tarafından geliştirilen ve "chatbot + yazılım"ı birleştiren bir yapay zekadır.

Kurulum:
1. İndirin
2. Gereksinimler dosyasından kütüphaneleri indirin
3. .env dosyasındaki "Şifrenizi girin" bölümünü kendi benzersiz şifrenizle değiştirin
4. Komut istemini açın ve "python main.py" komutunu çalıştırın
5. Açılan pencerede, 3. adımda belirlediğiniz şifreyi girin ve artık sohbet etmeye ve kod yazmaya hazırsınız. Yardıma ihtiyacınız olursa, "yardim" komutunu kullanın.

NOT: Sadece ingilizce harfler kullanilmalidir.
